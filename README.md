# Guitar Pedals!

### Digital guitar pedals using the STM32-F41xxx architecture.

## Goals/Todo:

- Write firmware algorithms for following effects:
  - Pure/Clean
  - Tuner
  - Boost
  - Delay
  - Overdrive/Distortion
  - Loop
  - Chorus
- Model and Create hardware
  - I/O jacks
  - Making attractive enclosure
  - Potentiometer for fiddling with the settings
  - Op Amp circuits for boosting and reducing impedance
